    markdown表格

    语法格式如下：

    |  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

    语法要点：

    • 表头与数据行之间要有分割线，至少3个连字符---
    • 两端竖线|可选，建议保留
    • 不需要严格对齐
    对齐方式：
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
    左对齐 右对齐 居中对齐
    单元格 单元格 单元格
    单元格 单元格 单元格 
    表格支持markdown大部分语法：
| 功能 | 描述 | 状态 |
|------|------|:----:|
| **用户登录** | 支持邮箱和手机号登录 | &#x2705; |
| *密码重置* | 通过邮箱重置密码 | &#x26a0;&#xfe0f; |
| `API接口` | RESTful API 设计 | &#x2705; |
| [文档链接](https://example.com) | 查看详细文档 | &#x1f4d6; |
    功能 描述 状态
    用户登录 支持邮箱和手机号登录 ✅
    密码重置 通过邮箱重置密码 ⚠️
    API接口 RESTful API 设计 ✅
    文档链接 查看详细文档 📖

    单元格内换行可以使用<br>
